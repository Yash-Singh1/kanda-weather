# `kanda-weather`

Weather dashboard for the Kanda Hackathon.

## Usage

To build into the `dist/` directory, run:

```sh
npm run build
```

To start the server, run:

```sh
npm run start
```

To start a development server, run:

```sh
parcel watch src/index.html
```

Then open a new terminal and run:

```sh
npm run dev
```
